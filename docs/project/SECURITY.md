# Security Policy

## Privacy model

Reader is designed to process documents locally on the user's Windows computer. It does not intentionally upload source documents, OCR text, or generated Markdown to external services.

## Supported versions

The latest commit on `main` is the supported development version. Public portable builds should be published through GitHub Releases.

## Reporting a vulnerability

Please open a GitHub issue if the problem can be described without sharing private documents.

If the report requires sensitive files or private details, do not post them publicly. Open a minimal public issue first and state that private details are available on request.

## Sensitive data

Never attach real private PDF/DOCX files to public issues. Use anonymized reproductions or small synthetic fixtures.

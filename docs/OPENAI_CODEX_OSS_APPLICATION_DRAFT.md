## Why this repository qualifies

Reader is a young but practical open-source Windows tool for private local conversion of PDF/DOCX documents to Markdown. It focuses on Russian OCR, scanned PDFs, mixed text/OCR PDFs, and users who cannot install Python. It helps legal and document-heavy workflows keep sensitive files local while producing searchable Markdown.

## How API credits would be used

API credits would be used for open-source maintenance: automated review of pull requests, security/code-quality checks, release notes, issue triage, and improving OCR/document-conversion tests. The tool itself will continue to process user documents locally by default.

## Additional note

Reader is designed for privacy-sensitive workflows. It does not intentionally upload source documents or extracted text. The project is being prepared with OSS documentation, CI, release workflow, issue templates, and a portable Windows build process.

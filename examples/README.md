# Examples

This folder contains synthetic examples only. They are not real legal, medical, financial, or personal documents.

The sample output shows the Markdown structure Reader creates after processing a folder with PDF/DOCX files:

- `sample_result/00_ALL_DOCUMENTS.md` - combined Markdown file.
- `sample_result/01_markdown/` - per-document Markdown files.

Do not add real private documents to this folder.

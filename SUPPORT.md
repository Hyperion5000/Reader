# Support

Reader is maintained as a small open-source project.

## Best Places To Ask

- Bug reports: open a GitHub issue with the bug report template.
- OCR quality problems: open an issue with the OCR quality template.
- Feature requests: open an issue with the feature request template.

## Privacy Rule

Do not attach real private documents to public issues.

Use:

- anonymized files;
- synthetic examples;
- screenshots with private data removed;
- short text excerpts that do not contain personal data.

## What To Include

- Windows version.
- Reader version or commit.
- File type: PDF text, PDF scan, mixed PDF, or DOCX.
- Expected result.
- Actual result.
- Whether `02_problem_files` was created.

## Not Supported

- Cloud processing of private documents.
- Direct support for old `.doc` files.
- Guaranteeing OCR accuracy for legally important text without manual review.
